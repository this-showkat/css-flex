Note(s):

  • To get benefits of flex, we must initialize flex with display: flex; or display: inline-flex; first. Then other attributes of flex (on container and/or item level) would be applicable.
  • Once display: flex; or display: inline-flex; is set, the default values for each property (for both of container and item) are applied by default.
  • In flex container, flex-flow could be used as shorthand for flex-diretion and flex-wrap. Example: flex-flow: row wrap;
  • The flex-wrap: nowrap; property prevents align-content: <value> from having an effect.

Note(s):

  • In flex item, flex coule be used as shorthand for flex-grow, flex-shrink and flex-basis. Example: flex: 0 1 auto;
  • Authors are encouraged to control flexibility using the flex shorthand rather than with flex-grow, or flex-shrink, or flex-basis directly, as the shorthand correctly resets any unspecified components to accommodate common uses.(~w3.org)
